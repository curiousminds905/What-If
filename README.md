# What If?

Explore science questions interactively in Space, Physics, Chemistry, and Biology.

---

## About

**"What If?"** is an educational website that allows users to explore curiosity-driven science questions.  
Each question shows **what is currently known** and **what remains unknown**, inspiring critical thinking, learning, and scientific exploration.  

The website is designed to make science **interactive, engaging, and fun** for students and enthusiasts alike.
